PyGame Intro

PyGame is a common introductory library for game development with Python. PyGame makes it easy to interface with a GUI, listen to key presses, identify mouse movements, and much more just with a few simple function calls. Similar to turtle graphics, this library allows for drawing content on a screen but with less hassle. With PyGame, one can create games as simple as complicated as terraria to as simple as snake.
